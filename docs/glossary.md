# Glossary

- **AOV**: Arbitrary Output Variable. Extra render passes beyond beauty.
- **Beauty**: The main rendered image (often `C` or `RGBA`).
- **Planes**: Named channel groups inside an EXR.
- **OIDN**: Intel Open Image Denoise (CPU).
- **OptiX**: NVIDIA OptiX denoiser (GPU).
- **Temporal**: Uses previous frame information to improve stability.
- **idenoise**: Houdini's command-line denoiser executable.
